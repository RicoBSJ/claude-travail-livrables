Effectue une veille des publications récentes de la HAS (Haute Autorité de Santé).

1. Consulte le site https://www.has-sante.fr et recherche les publications récentes pertinentes pour le secteur médico-social (ESSMS : foyer de vie, FAM, MAS, SAVS, SAMSAH).
2. Filtre les documents liés aux thématiques suivantes : RBPP, bientraitance, autodétermination, TNmP, TDI/TSA, QVT, droits des personnes accompagnées, maltraitance.
3. Pour chaque publication pertinente trouvée, rédige un résumé structuré d'une page maximum :
   - Titre complet et date de publication
   - Public cible
   - 5 à 7 points clés (bullet points)
   - Lien vers le document original
4. Sauvegarde le résumé dans : `Veille/` avec le nom de fichier `YYYY-MM-DD_veille_HAS.md` (date du jour).
5. Si un fichier de veille existe déjà pour aujourd'hui, demande confirmation avant d'écraser.
6. Confirme en affichant le nombre de publications trouvées et les titres retenus.
